## Troubleshooting

### Logo Not Displaying
- Verify file exists in `/public/bhhs-logo.png`
- Check file name matches exactly (case-sensitive)
- Ensure file is a valid image format (PNG, JPG, SVG)
- Clear browser cache and reload page

### Headshot Not Displaying
- Verify file exists in `/public/dr-jan-duffy-portrait.jpg`
- Check file name matches exactly (case-sensitive)
- Ensure file is a valid image format (JPG, PNG)
- Clear browser cache and reload page

### Image Quality Issues
- Ensure images are high-resolution
- Optimize file sizes for web
- Use appropriate image formats (PNG for logos, JPG for photos)
- Test on different devices and screen sizes
